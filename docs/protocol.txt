# 子协议说明

////////////////////////////////////////////////////////////////////////
通用字段

int
包字节长度，使接收方明确知道需要接受多少数据

unsigned short
子协议类型，不论接收或者发送都要在消息前方加协议类型

例：
包长度 + 协议 + 首部 + 正文

返回值：
包长度 + 协议 + 状态 + 正文

////////////////////////////////////////////////////////////////////////

正文

int flag
